Which Version
Do You Want?

This page is now just of historical interest. See the download page for a summary of the various versions of E.

 

  • The current release of E, 0.8.24, is about twice as fast.

  • In 0.8.23, capDesk works again.

  • 0.8.22 supports a limited form of persistence and upgrade. 16 bugs closed!

  • 0.8.12. is the release subjected to a security review by David Wagner and Dean Tribble.

  • tl-otc-0.8.9.1b was a previous important version of E, since it was the last/best one with a functioning 3vat CapTP system. Users of this should try moving to a more current release as soon as they can afford the conversion costs, and if they don't currently need support for 3vat introduction. If you need help, please contact me, the webmaster-at-erights.org.

I only maintain links above for a small number of the most recent releases, and for certain important old versions so long as there's a reason why someone might want to use these. If you wish to obtain a version for which there's no link above, please let me, the webmaster-at-erights.org, know.
