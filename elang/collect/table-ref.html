Messages Shared with all Tables

Since these messages are shared between Const and Flex tables, they must be query only. As applied to a Flex table, they operate of the state of the table at that instant.

Since they are shared between lists and maps, they must be have a common meaning, whether an index is used with a list, or a key is used with a map.

Since "vice" maps presidents to their vice presidents, "clinton" is a key of the list (since he was a president), but is not present as a value (since he was never a vice president).
